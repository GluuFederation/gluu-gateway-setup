# Node JS Demo API

## Pre-requisites

1. NodeJS > v10.x.x and NPM > 6.x.x

## Installation

1. `cd node-api-2`
1. `npm install`

## Start

```
npm start

or

node index.js
```

It will start node js server on port `3000`

## APIs

There are `company` and `item` apis available with dummy data.

Request `GET http://localhost:3000/company` to get companies.
